Configure Git

  1. GO to vs code and click terminal arrow then new terminal
  2. Check if git is already installed by typing git --version in the command line
  3. Configure git in your local conputer
    • Set your Name -> git --config global user.name "your name"
    • Set your Email Address -> git --config global user.email "your email"
  4. To initialize the project directory type git init

Git Add and commit

  1. To see the status of the file in your directory type git status
  2. To track new files use the command git add
    For group files, git add .
    For individual, git add "file name"
  3. By running the git status again, you can see the Staged files in green ready for commit.
  4. Commit the files that you have staged in your local repository by typing git commint -m
